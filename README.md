## Usage

> 1) fork the repo
> 2) clone from your fork into your local machine
> 3) add environment variables
> 4) npm install
> 5) npm run build
> 6) npm start
> 7) enjoy

## Requirements

- Node 9.2.1
- MySQL 14.14
- npm 5.6.0

## Development

### Installing Dependencies

From within database directory:

```sh
mysql -u <username> -p < createDB.sql
```

From within the root directory:

```sh
npm install
```
